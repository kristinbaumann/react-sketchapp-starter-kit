## Usage

* Clone this repository
    ```bash
    git clone https://github.com/kristinbaumann/react-sketchapp-starter-kit project-name
    cd project-name
    rm -rf .git

* Install npm dependencies
    ```bash
    npm install
    ```

* Open Sketch with a new project

* Render the React code in Sketch
    ```bash
    npm run render
    ```
